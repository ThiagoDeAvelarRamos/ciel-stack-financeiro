# Cieal Stack Financeiro

Este projeto é uma aplicação web para controle de receitas e despesas pessoais. Ele permite que os usuários registrem suas transações financeiras e visualizem um resumo de suas finanças.

## Estrutura do Projeto

O projeto possui a seguinte estrutura de arquivos:

```
ciel-stack-financeiro
├── css
│   ├── globals.css          # Cores da página definidas como variáveis CSS
│   ├── header.css           # Estilos específicos para o cabeçalho
│   ├── summary.css          # Estilos para a seção de resumo
│   ├── transaction-form.css  # Estilos para o formulário de entrada
│   └── transaction-history.css # Estilos para o histórico de transações
├── index.html               # Estrutura HTML principal do projeto
└── README.md                # Documentação do projeto
```

## Instalação

1. Clone o repositório:
   ```
   git clone <URL_DO_REPOSITORIO>
   ```
2. Navegue até o diretório do projeto:
   ```
   cd ciel-stack-financeiro
   ```

## Uso

1. Abra o arquivo `index.html` em um navegador web.
2. Utilize o formulário para adicionar novas transações.
3. Visualize o resumo e o histórico de transações na página.
